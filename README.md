## Description
   Social Network API is an API for a social network web application where users can share their thoughts, react to friendsâ€™ thoughts, and create a friend list. It features the use of Express.js for routing, a MongoDB database, and the Mongoose ODM. 

## Installation
    Run npm install and npm run seed in the terminal.
    
## Usage
    Run start in the terminal. Test API routes in Insomnia Core to create, update, and delete data in the database.
